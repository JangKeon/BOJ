# Dynamic Programming

N X N 크기의 Map 에 숫자들이 주어지고 좌표 A부터 좌표 B까지의 합을 구하는 문제이다.

하지만 일반적인 방법으로 이중 for문을 돌리면서 정답을 구하려 하면 당연히(?) 시간초과가 난다.

따라서 Map에 숫자들을 입력받고 각 열에 대한 합을 저장해놓는 배열을 하나 더 만든다.

이후 좌표 두 개를 입력 받고 Y1~Y2까지 반복하며 sum_row[X2][i] - sum_row[X1-1][i] 를 결과에 더해주면 된다. (여기서 X와 Y의 순서가 바뀌는 이유는 입력받는 좌표가 사실상 Y, X 순으로 들어오기 때문이다.)
